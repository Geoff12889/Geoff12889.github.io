# My Portfolio

## Description
This my GitHub hosted portfolio. It was created as my final project for GIT-215. My portfolio website has [five linked pages](#pages): the homepage, about me, resume, projects and contact page. The website was designed with mobile-first principles. It is responsive in that it fits a small mobile screen and grows to utilize a full desktop screen. The HTML is validated. There is currently no JavaScript.
