# My Portfolio

## Description
This is my personal portfolio website hosted on GitHub Pages, created as my final project for GIT-215. The site includes five connected pages: Home, About, Resume, Projects, and Contact.

The portfolio was designed using mobile-first development principles and responsive design techniques. It adapts from smaller mobile screens to larger desktop displays while maintaining a consistent layout, typography, and navigation structure.

This project focuses on semantic HTML, responsive CSS, accessibility, and modern layout techniques including CSS Grid and Flexbox.
